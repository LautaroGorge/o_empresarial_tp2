## ESCENARIO ELEGIDO
* **Escenario B:** Análisis de Ventas de una Pequeña Empresa.

## DESCRIPCIÓN DEL DATASET
Este proyecto utiliza un conjunto de datos simulados de ventas comerciales en formato CSV. Contiene registros con información clave como identificador de transacción (`id`), fecha de venta (`sales_date`) y el monto de la operación (`sales_amount`), lo cual permite calcular indicadores de desempeño de la empresa.

## INSTRUCCIONES DE EJECUCIÓN
1. Clonar el repositorio.
2. Asegurar que el archivo de datos esté ubicado en la carpeta `/datos`.
3. Ejecutar el script `analisis_datos.py` ubicado en la carpeta `/scripts`.
4. Los resultados y gráficos correspondientes se exportarán automáticamente en la carpeta `/resultados`.
